# simple_shell
My second group Project at ALX-school

This is a simple shell program  concieved  to run on Ubuntu 22.04 LTS


Functions

shell.h :  this is my custom header file to make my shell program work ,
  it contains the definitions of the functions supported and the parameters which they need to run.

_atoi.c : Defines some useful characters
builtin.c :  Helps to manage directories
errors.c : Prints errors and strings
exiter.c : A function to manage stings
get_line.c : To manage input from standard input 
history.c : To keep track of the shell run instances
lists.c : To handle all list elements
more_errors.c : More functions to handle errors
more_string.c : More functions to handle strings
 
 
shloop.c :  This defines the looping conditions of the shell program
vars.c :    this file takes care of handling chains and tokens.
wami.c . To manage aliases
environ.c : To manage the environment parameters 
getinfo.c :  To manage info variables
main.c :  A main function to define an entry point for file management
more_lists.c: More functions to manage lists
parser.c: Functions to help with parsing
realloc.c:  To help with memory management
string.c:  Defines basic string manipulation functions
tokengen.c: To help to  generate tokens.
memory.c : To help free memory
